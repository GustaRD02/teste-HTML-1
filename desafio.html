Comunicação Client X Serverr

(Voltar)

Na comunicação client x server na web, um navegador (cliente) faz solicitações a um servidor web para acessar recursos, como páginas da web, imagens ou dados. O servidor responde fornecendo esses recursos. Essa troca de solicitações e respostas é fundamental para o funcionamento da internet, permitindo a interação entre outros usuários e servidores para visualização de conteúdo online, processamento de formulários, entre outras atividades. Protocolos comuns usados nesse contexto incluem HTTP (Hypertext Transfer Protocol) e HTTPS (HTTP Secure).

Client

Na web, um "cliente" refere-se a um programa de computador, geralmente um navegador da web, que solicita serviços ou recursos a partir de um servidor. O cliente envia requisições para o servidor, como pedidos de páginas da web, dados ou imagens e recebe as respostas correspondentes. É o componente que os usuários interagem diretamente para visualizar conteúdo online e interagir com aplicativos web. Os navegadores, como Chrome, Firefox e Safari, são exemplos comuns de client na web.

Server

Um "servidor" na web é um computador ou programa de software que fornece serviços, recursos ou dados para outros dispositivos chamados clientes. Em um contexto web, os servidores armazenam e gerenciam páginas da web, aplicativos e outros conteúdos, respondendo às solicitações dos clientes. Eles desempenham um papel crucial no fornecimento de informações e serviços pela internet, garantindo que os usuários possam acessar e interagir com os recursos disponíveis. Exemplos de servidores web como Apache, Nginx, e servidores de aplicativos que hospedam sites dinâmicos.

O que é programação

(Voltar)

Linguagem de programação é um conjunto de regras e instruções utilizadas para escrever software ou programas. Essas linguagens permitem que os programadores comuniquem suas intenções para um computador de maneira que a máquina possa entender e executar as tarefas desejadas. As linguagens de programação podem variar em complexidade e finalidade, sendo utilizadas para desenvolver uma ampla variedade de aplicativos, desde um simples script até programas complexos e sistemas. Exemplos como Python, Java, C++, JavaScript, C#, PHP, entre muitas outras.*

Tags

(Voltar)

As tags são elementos de marcação que envolvem o conteúdo em uma página web, fornecendo instruções sobre como esse conteúdo deve ser apresentado ou comportar-se. Aqui estão algumas tags HTML.

html

Define o início e o fim do documento HTML.

head

É como se fosse a cabeça da página, contém informações sobre o documento, coisas que ele carrega antes de mostrar ao usuário.

tittle

Umas das tags usadas dentro da tag head, que é usada para o nome/título da página, exibido na barra de título do navegador.

body

É como se fosse o corpo da página, o conteúdo visível da página, o que vai aparecer para o usuário.

h1

É um exemplo de cabeçalho, um título da sua página, tendo outros como h2, h3, h4 e assim consecutivamente até o h6, se tornando subtítulos dos subtítulos.

p

É cada parágrafo, é os textos, um corpo, um texto novo.
